早起きバトル（仮）


寝てる間はキャラクターも寝ている。先に起きて、寝坊した相手を攻撃せよ。



概要

「早起き × 対戦ゲーム」アプリ。プレイヤーは前夜に翌朝の起床時刻を宣言し、同じく宣言した相手とマッチングされる。朝、宣言した時刻の前後にアクション（ボタン操作）を起こして相手を攻撃する。早く起きてアクションを起こした方が勝ち、寝坊した方は攻撃を受けて負ける。 勝敗はポイントとして蓄積され、ランキングを競う。

「ゲームに勝ちたいから早く起きる」という動機づけで、早起きの習慣化を後押しすることを狙う。

コンセプト


寝ている間は、自分のゲームキャラクターも寝ている状態
先に起きないと、起きた相手から一方的に攻撃を受けて負ける
「起きること」そのものがゲームのアクションになる


主な機能


アカウント作成 … プレイヤー登録・ログイン
起床時刻の宣言 … 毎晩22時までに、翌朝の起床予定時刻を設定する
マッチング … 22時の時点で対戦相手を決定し、対戦を成立させる

初期段階では「全員がマッチングできる」前提で実装
拡張として、相手が見つからない場合のコンピュータ（CPU）対戦パターンを検討



バトルフィールド … 対戦の進行・状態を表示する画面
攻撃アクション … 相手を攻撃するためのボタン操作（スマホを振る／文字を打つ など）
ポイント／ランキング … 勝敗に応じてポイントを付与し、ランキングを表示


ゲームの流れ


宣言（〜前夜22時）
プレイヤーが翌朝の起床予定時刻を設定する。
マッチング成立（22時）
宣言したプレイヤー同士で対戦相手が決まる。
バトル（翌朝）
宣言した時刻の前後に設けられた「攻撃可能ウィンドウ」内で、アクションを起こして攻撃する。
決着
先に起きてアクションした側が勝利。寝坊してウィンドウを逃した側は敗北。
ポイント反映・ランキング更新


攻撃可能ウィンドウ

攻撃ボタンを押せるのは、宣言した起床時刻を基準とした一定時間のみ。


開始：宣言時刻の 10分前
終了：宣言時刻の 1時間後



早すぎる先押し・遅すぎる寝坊のどちらも無効にすることで、「宣言した時刻ぴったりに起きる」インセンティブを作る。時間幅は調整前提のたたき台。
